# SeeSea-Backend

![avatar](C:\Users\Germi\Desktop\未命名.png)

第八屆菁英軟體工程師戰鬥營期末專題─「SeeSea」，是一款潛水揪團App，提供多元的潛水活動及在地化的潛水資訊，讓使用者能夠迅速找到適合自己的潛水活動。此專案為App的後端Web API，在GCP建立Linux虛擬機供前端使用。
